# CAN FD 联调协议

当前协议仅用于 STM32G474 固件内部回环和首轮硬件联调，不是最终车端控制协议。

## 总线参数

- 帧格式：CAN FD，开启 BRS。
- 仲裁速率：500 kbps。
- 数据速率：2 Mbps。
- 标识符：11 位标准 ID。

## 回环报文

| 项目 | 值 |
|---|---|
| CAN ID | `0x321` |
| 方向 | STM32 自发自收 |
| DLC | 8 bytes |
| Payload | `43 48 41 53 53 49 53 01` |
| ASCII | `CHASSIS` + 版本字节 `0x01` |

该报文仅验证 FDCAN 初始化、过滤器、中断、发送和接收链路。接入 Jetson 前需要重新定义控制指令、状态上报、故障码、计数器和超时策略。
